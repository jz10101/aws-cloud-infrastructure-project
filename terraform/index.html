AWS Cloud Infrastructure Project

Built and deployed by Jose Fernandez using Terraform, AWS EC2, Linux, Apache, and Infrastructure as Code.

🟢 Live Infrastructure • Terraform Managed • Public AWS Deployment

Project Summary

This project demonstrates how modern cloud infrastructure can be deployed entirely with Terraform on AWS. Every major AWS resource was created using Infrastructure as Code instead of manual configuration.

Infrastructure Includes

  • Amazon EC2 Linux Server
  • Custom VPC
  • Public Subnet
  • Internet Gateway
  • Route Table
  • Security Group
  • Terraform Infrastructure
  • Apache Web Server

Skills Demonstrated

  • Infrastructure as Code (Terraform)
  • AWS Networking
  • Linux Administration
  • SSH Remote Management
  • Cloud Deployment
  • Infrastructure Troubleshooting

Next Phase

This infrastructure will soon be upgraded from Apache hosting to a Dockerized Nginx container as part of the next deployment phase.
