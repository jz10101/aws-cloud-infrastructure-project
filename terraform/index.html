About This Project

Production-style infrastructure built through code

This project demonstrates how cloud infrastructure can be designed, deployed, and managed through Infrastructure as Code using Terraform on Amazon Web Services.

The application is hosted on an Amazon EC2 Linux server, runs inside a Docker container, uses IAM for controlled access, stores project assets in Amazon S3, and is monitored with Amazon CloudWatch.
